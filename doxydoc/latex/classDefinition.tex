\doxysubsection{Detailed Description}
The common base class of all entity definitions found in the sources.

This can be a class or a member function, or a file, or a namespace, etc. Use \mbox{\hyperlink{classDefinition_a526c39074637d4b17a3f67df56d961ff}{definition\+Type()}} to find which type of definition this is.
