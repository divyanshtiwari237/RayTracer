\doxysubsection{Detailed Description}
Singleton for configuration variables.

This object holds the global static variables read from a user-\/supplied configuration file. The static member \mbox{\hyperlink{classConfigImpl_a501e98a30e96c9930ac4b1791b80c09a}{instance()}} can be used to get a pointer to the one and only instance.

Set all variables to their default values by calling Config\+::instance()-\/\texorpdfstring{$>$}{>}\mbox{\hyperlink{classConfigImpl_a50654e77279eecd43b0dd91cb21420e5}{init()}}
