\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Symbol\+Map$<$ T $>$}
Class implementing a symbol map that maps symbol names to objects. Symbol names do not have to be unique. Supports adding symbols with \mbox{\hyperlink{classSymbolMap_a5ee5036f5adbd147a02e4aa476fc8b72}{add()}}, removing symbols with \mbox{\hyperlink{classSymbolMap_a36b000d1d74ec4820b0d1c3a57dd645c}{remove()}}, and finding symbols with \mbox{\hyperlink{classSymbolMap_abd4aed07bc7ac9abd15f1046875a4698}{find()}}.
