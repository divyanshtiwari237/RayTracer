\doxysubsection{Detailed Description}
Variant class for a scoped type.

Variants\+:
\begin{DoxyItemize}
\item Dummy\+: a type used for hiding a global type.
\item Local\+: a locally defined type (e.\+g. found inside a function)
\item Global\+: a globally defined type (processed by doxygen in an earlier pass). 
\end{DoxyItemize}
