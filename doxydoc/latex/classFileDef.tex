\doxysubsection{Detailed Description}
A model of a file symbol.

An object of this class contains all file information that is gathered. This includes the members and compounds defined in the file.

The member \mbox{\hyperlink{classFileDef_a09f77095e41c7b6c9f9b9e54fb702dcb}{write\+Documentation()}} can be used to generate the page of documentation to HTML and La\+TeX.
