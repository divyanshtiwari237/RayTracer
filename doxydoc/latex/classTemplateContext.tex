\doxysubsection{Detailed Description}
Abstract interface for a template context. 

A Context consists of a stack of dictionaries. A dictionary consists of a mapping of string keys onto \mbox{\hyperlink{classTemplateVariant}{Template\+Variant}} values. A key is searched starting with the dictionary at the top of the stack and searching downwards until it is found. The stack is used to create local scopes. \begin{DoxyNote}{Note}
This object must be created by \mbox{\hyperlink{classTemplateEngine_abd065687312d4607d538e5a84d91e8fe}{Template\+Engine\+::create\+Context()}} 
\end{DoxyNote}
