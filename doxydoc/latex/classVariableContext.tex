\doxysubsection{Detailed Description}
Represents a stack of variable to class mappings as found in the code. Each scope is enclosed in \mbox{\hyperlink{classVariableContext_ae1d858d3ad4778e79ac69a021474edab}{push\+Scope()}} and \mbox{\hyperlink{classVariableContext_a648ebfb9146a61b70dffe684aec77940}{pop\+Scope()}} calls. Variables are added by calling add\+Variables() and one can search for variable using \mbox{\hyperlink{classVariableContext_aa4cfe2b318b52c33b4e28b5840275d85}{find\+Variable()}}.
