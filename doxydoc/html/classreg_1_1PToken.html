Detailed Description

Class representing a token in the compiled regular expression token stream. A token has a kind and an optional value whose meaning depends on the kind. It is also possible to store a (from,to) character range in a token.

Member Enumeration Documentation

◆ Kind

enum class reg::PToken::Kind : uint16_t
strong

The kind of token.

Ranges per bit mask:

  • 0x00FF from part of a range, except for 0x0000 which is the End marker
  • 0x1FFF built-in ranges
  • 0x2FFF user defined ranges
  • 0x4FFF special operations
  • 0x8000 literal character
